Управляющие структруы в Go
Оператор for (цикл с параметром) 
							 for init; condition; inc { 
							 statements 
							 }
Оператор if 
							 if condition { 
							 // do some action 
							 } else { 
					         // do some action 
							 }
Оператор switch 
							 switch expression { 
							 case 1: // do some action 1 
							 case 2: // do some action 2 
							 case 3: // do some action 3 
							 default: // do default action 
							 }
