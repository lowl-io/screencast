Тестирование в Go
Тестирование
Язык Go имеет поддержку автоматического тестирования пакетов. Протестироват пакет можно при помощи команды go test. Данная команда запустит функцию, которая имееет следующий вид 
							 func TestXxx(*testing.T)
где Xxx - это любая буквенно-цифровая строка. (Test - не может быть изменено)
Для того чтобы написать новый тест, нужно создать файл с окончанием _test.go, который содержит функцию TestXxx, как было рассмотрено выше.
