Функции

Переменное число аргументов функции
Использование ... перед типом последнего аргумента означает, что функция может содержать ноль и более таких параметров. В нашем случае мы берем ноль и более int. 
								
				func add (args ...int) int {
				    total := 0

				    for _, v := range xs {

				    }
				    return total
				}

				func main() {

				    fmt.Println(add(1,2,3))
				}
Замыкания
Замыкание(если очень просто) - это возможность создавать функции внутри функциий 
							
				func main() {
				  x := 0
				  inc := func() int {
				      x++
					  return x
				  }

				  fmt.Println(inc()) // 1
				  fmt.Println(inc()) // 2
				}
Рекурсия
Рекурсия - это когда функция вызывает сама себя 
								
			  func fact(x uint) uint {

			  	  if x == 0 {

				    return 1
									
				  }

				  return x * fact(x-1)
			  }
