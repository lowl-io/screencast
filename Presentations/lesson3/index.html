Переменные в Go
Переменная - это именнованое место хранения какого-то типа данных.
Создание перменных
x := "Hello World"
var x string = "Hello World"
var x = "Hello World"
Как назвать перменную
Имена перменных в Go должны начинаться с буквы и могут содеражть
буквы, цифры и знак подчеркивания

myNameIs x_ y slide_4 qwerty123_

Константы
Константа - это величина, чье значение не может быть изменено после инициализации
const x string = "Hello World"
